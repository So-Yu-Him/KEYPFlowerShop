Lending Flow (Conceptual)

This sequence illustrates a possible contract-driven lifecycle for demo and classroom discussion

📋

Borrower Requests Loan ($5,000 scenario — 100 PLDT tokens)

Borrower onboarding, identity checks, and credit scoring are presented as conceptual product steps rather than live external integrations.

💰

Lenders Fund the Loan ($50 per PLDT token in demo pool)

Each $50 contribution is tracked in a simulated pool. At 100% funding, the UI demonstrates disbursement, platform fee allocation, and a 12-month repayment timeline.

📅

Monthly Repayment Updates (~$433.68 / month scenario)

Repayments are represented in UI based on the chosen schedule. Oracle confirmation and on-chain distribution are conceptual extension points, not active integrations.

✅

Loan Fully Repaid — Tokens Burned / Default Protocol

Full repayment scenario: each lender receives $54 per $50 token (~8% APR). Token burn, closure, and default handling are simulated outcomes shown for product discussion.

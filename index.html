Smart Contract Flow

Every dollar is automatically distributed on-chain — no middlemen, full transparency

🛒

Buyer Purchases Token ($15 held in escrow)

$15 locked in the smart contract. Buyer's public key passes AML screening via Chainalysis. BLMT token minted and sent to buyer's wallet.

🌹

Token Redeemed for Flowers

Smart contract immediately releases: $5 → flower supplier, $2 → courier. Remaining $8 stays in escrow pending delivery.

📦

Oracle Confirms Delivery

Courier marks delivery complete off-chain. Chainlink oracle triggers the smart contract. $7.55 released to florist, $0.45 platform fee retained.

Token Expires After 12 Months

If unredeemed, the contract auto-releases $14.55 to the florist and retains $0.45 platform fee. No manual action required.
